## Installation and Setup

1. Clone the repository
```bash
git clone https://github.com/yourusername/portfolio-project.git
```

2. Install dependencies
```bash
cd portfolio-project
npm install
```

3. Run the development server
```bash
npm start
```

4. Build for production
```bash
npm run build
```
